## Learning Objectives

- Install Tailscale on at least two devices
- Authenticate both devices to the same tailnet
- Verify connectivity with `tailscale ping` and `tailscale status`
- Understand the `tailscaled` daemon and `tailscale` CLI relationship
- Know how to add, remove, and expire devices from the admin panel
